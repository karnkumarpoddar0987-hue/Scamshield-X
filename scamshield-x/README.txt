============================================
  ScamShield X — How to Run
============================================

PROBLEM: Agar aap index.html directly browser mein open karte ho
(file:// se), toh backend kaam nahi karega.

SOLUTION — 2 tarike:

--- Option 1: Double-click (Easiest) ---
  START_HERE.bat file ko double-click karo.
  Automatically server start hoga aur browser open hoga.

--- Option 2: Terminal se ---
  1. Terminal/CMD open karo
  2. Is folder mein jao:
     cd "C:\Users\Administrator\Desktop\Codefussion  PROJECT\scamshield-x"
  3. Run karo:
     node server.js
  4. Browser mein open karo:
     http://localhost:3000

============================================
  IMPORTANT: ALWAYS open http://localhost:3000
  NEVER open index.html directly
============================================
